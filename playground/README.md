# Playground

This directory contains various POCs and experimental projects for the hackathon.

## How to Use

1. Create a new directory for your POC: `mkdir my-poc-name`
2. Add a README.md explaining your POC
3. Develop and experiment freely

## POC Template Structure

```
my-poc/
├── README.md           # What this POC does
├── src/               # Source code
├── tests/             # Tests (optional)
└── package.json       # Dependencies (if applicable)
```
