# Gemini エージェントのオンライン学習に適したゲーム環境の定義

> 調査日：2026-02-21（Gemini 3 Tokyo Hackathon 当日）

## 背景

Gemini チームはゲーム環境をエージェントのオンライン学習に活用したい意向を持っている。
本ドキュメントは、どのような環境がその方向性に合致するかを、DeepMind の公開論文・ブログから調査したものである。

## 中心的な研究：SIMA 2（2025年12月）

- 論文：[arxiv.org/html/2512.04797v1](https://arxiv.org/html/2512.04797v1)
- ブログ：[SIMA 2: A Gemini-Powered AI Agent for 3D Virtual Worlds](https://deepmind.google/blog/sima-2-an-agent-that-plays-reasons-and-learns-with-you-in-virtual-3d-worlds/)

### 概要

SIMA 2 は Gemini を推論コアとして使い、3D ゲーム環境の中でエージェントが**自律的に自己改善**するシステム。
人間によるデモデータなしに、Gemini が報酬を生成することでオンライン学習が可能。

## 環境が満たすべき要件

| 要件 | 詳細 |
|------|------|
| **視覚的インターフェース** | エージェントはスクリーンを見て操作（ゲーム内部 API への直接アクセスなし）|
| **標準的な入力** | キーボード・マウス相当の操作（人間と同じ UI）|
| **タスクの検証可能性** | 成功/失敗を自動で判定できる（OCR・ピクセル・スコア等）|
| **多様なタスク** | 収集・製作・戦闘・探索など複数の行動タイプ |
| **オープンエンドな設計** | 事前定義された報酬なし → Gemini が報酬を生成する余地がある |

## 自己改善ループの仕組み

```
Gemini がタスクを生成
      ↓
エージェントがゲームをプレイ（試行錯誤）
      ↓
Gemini がプレイを Rubric で 0〜100 スコアリング
      ↓
経験データをバンクに蓄積
      ↓
次のトレーニングに再利用（人間データ不要）
```

Gemini による報酬モデルは「タスク完了度」と「行動の目的合致性」を5件の独立評価で測定する。

## 関連プロジェクト

| プロジェクト | 概要 | 関連性 |
|-------------|------|--------|
| **SIMA 2** | Gemini 搭載の汎用ゲームエージェント | 最も直接的な参照元 |

## Gemini が目指すエージェント学習の3つの柱

最新論文（SIMA 2 / Gemini 2.0 Technical Report）で重要視されている要素：

### 1. Multimodal Interactive Learning（多峰性インタラクティブ学習）
テキスト指示だけでなく、ビデオ（視覚）・音声・画面へのスケッチ（Visual Prompting）を理解して行動を修正する能力。

### 2. Open-ended Self-Improvement（オープンエンドな自己改善）
エージェントが自ら新しいタスクを生成し、報酬も Gemini 自身が判定して学習サイクルを回す（Self-Reward）。

### 3. Zero-shot Adaptation to New Worlds
見たことのない新しいゲームや物理法則の環境に放り込まれた際、即座に探索して適応する能力。

## ハッカソンで作るゲームへの示唆

Gemini チームが「使いたい」と思う環境の条件：

1. **Gemini がタスクを自動生成できる** — 固定シナリオでなく動的に目標が変わる
2. **Gemini が報酬を判定できる** — スクリーン観察だけで成否を評価できる
3. **視覚的状態がリッチ** — スクリーンショットから意味が読み取れる
4. **人間もプレイできる** — 収集データの質保証 & デモ映え
5. **オープンワールド or 手続き生成** — 無限に多様な経験を供給できる
6. **指示の動的変更** — プレイ中にリアルタイムで優先タスクを変えられる
7. **多峰性フィードバック** — スクリーンショット＋テキスト説明の両方をエージェントに渡す
8. **Self-Reflection の余地** — 成否をシステムフラグだけでなくエージェント自身の推論で判定できる

## 理想的な環境アーキテクチャ案

```
┌─────────────────────────────────────────┐
│         Game Environment                │
│  - 自然言語タスク受付（API経由）          │
│  - スクリーンショット出力                 │
│  - 行動受付（keyboard/mouse 相当）        │
└──────────────┬──────────────────────────┘
               │ state（screenshot + text）
               ▼
┌─────────────────────────────────────────┐
│         Gemini Agent                    │
│  - 状況理解（multimodal）               │
│  - 行動選択（CoT）                      │
│  - 自己評価（Self-Reflection）          │
└──────────────┬──────────────────────────┘
               │ action + reasoning log
               ▼
┌─────────────────────────────────────────┐
│         Online Learning Log             │
│  - 思考(CoT)・行動・結果をデータセット形式で保存  │
│  - そのままファインチューニングに使える形式で出力  │
└─────────────────────────────────────────┘
               ▲
               │ human feedback（任意）
┌─────────────────────────────────────────┐
│         Human-in-the-loop               │
│  - 「その動きは危ない」等の介入           │
│  - エージェントがプロンプトを即時更新     │
└─────────────────────────────────────────┘
```
